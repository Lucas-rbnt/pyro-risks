pyro_risks.datasets¶

The datasets submodule contains datasets with entries indexed by location and datetime, to help prediction outbreak risks and wildfire evolution. Using those datasets means that you comply with the usage restrictions from the corresponding authors.

The following datasets are available:

Models

  • Region tiling

    • France

  • Weather data

    • France

  • Wildfire history

    • France

Region tiling¶

GPS coordinates often need to be mapped for later aggregation. These tools are meant to help with the aggregation process by defining sub-areas in a country or geographical area.
